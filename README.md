# ECOPHA Website Prototype

A modern website prototype developed for **ECOPHA**, an Australian sustainable packaging company, as part of an **RMIT University marketing campaign**. The project showcases ECOPHA's sustainability mission through an interactive digital experience featuring modern design, animations, and data visualizations.



## Features

* Responsive and interactive user interface
* Dark mode support
* Smooth page transitions and animations
* Interactive sustainability dashboards
* HoneyCat brand mascot integration
* Gamified sustainability experience
* Accessible UI components built with shadcn/ui

## Tech Stack

* **Frontend:** React 18, TypeScript, Vite
* **Styling:** Tailwind CSS v4, shadcn/ui
* **Animations:** Motion
* **Routing:** React Router v7
* **Charts:** Recharts
* **Design:** Figma
* **Package Manager:** pnpm

## Getting Started

### Install

```bash
npm install
```

or

```bash
pnpm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

## Project Context

This prototype was developed as part of the **Marketing for Managers (BUSM1534)** course at **RMIT University** in collaboration with **ECOPHA**. The website demonstrates a proposed digital presence to support the company's sustainability-focused marketing strategy.

## Disclaimer

This repository contains an academic prototype created for demonstration purposes. The content and features represent a proposed concept rather than a production-ready website.
